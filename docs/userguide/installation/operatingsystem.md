# Operating System Selection and Configuration

Pre-built packages are provided for the officially supported platforms listed on the [Installing on Linux](linux.md) page. If you build from source, a wide variety of operating systems are possible.

In addition to the officially supported platforms, the following systems have been community-tested:


* Rocky (8, 9)
* Alma (8, 9)
* Debian (11,12)
* Ubuntu (18, 20, 22, 24)
* OpenSuse Leap (15.4)
* Azure Linux (CBL-Mariner2)
* Amazon Linux 2
* Amazon Linux 2023
* Kali (2023.1)
* CentOS 7 (obsolete and not recommended)
* RHEL 7,8,9
* Mariner
* Linux Mint (Cinnamon)

KumoMTA works with x86 and Arm processors.

Note that most "dnf" systems work fine with the Rocky install [instructions](../installation/linux.md).
And that most "apt" systems work fine with the Ubuntu install [instructions](../installation/linux.md).
