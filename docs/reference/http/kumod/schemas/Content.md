The message content.
Can either be a fully formed MIME message, or a json
object describing the MIME structure that should be created.


Can be one of the following shapes of data:

## Option 1
A complete MIME message string

This is a string value.

## Option 2
Describe the MIME structure to be created

This is an object value, with the following properties:


  * `amp_html_body` - optional nullable `string`. If set, will be used to create a text/x-amp-html part

  * `attachments` - optional array of [Attachment](Attachment.md). Optional list of attachments

  * `from` - optional nullable [FromHeader](FromHeader.md). 

  * `headers` - optional `object`. Optional map of headers to include in the message.
    This is a map of header name to header value

  * `html_body` - optional nullable `string`. If set, will be used to create a text/html part

  * `reply_to` - optional nullable [FromHeader](FromHeader.md). 

  * `subject` - optional nullable `string`. Set the Subject: header

  * `text_body` - optional nullable `string`. If set, will be used to create a text/plain part
